GoodNum
=================

A small library that adds commas to numbers.

## Installation
  'npm install @hpaiz/goodnum'

## Usage
  var formatNum = require('@hpaiz/goodnum');
  var formattedNum = formatNum(35689);

  Output should be `35,689`

# Tests
